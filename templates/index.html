Bienvenidos al Sistema de Gestión de Ventas de Celulares

Este sistema ha sido desarrollado para gestionar de manera eficiente las operaciones de venta de celulares, permitiendo a los usuarios administrar diferentes aspectos del negocio desde un solo lugar.

Funcionalidades Principales

  • Gestión de Equipos: Permite agregar y modificar
  • Administración de Categorías: Gestión de categorías
  • Modelos y Fabricantes: Información detallada de modelos y fabricantes.
  • Control de Stock: Seguimiento de las cantidades disponibles y su ubicación.
  • Gestión de Proveedores: Información de contacto y detalles de proveedores.
  • Accesorios: Administración de accesorios compatibles con los equipos.

Cómo Usar el Sistema

Navega a través del menú superior para acceder a las distintas secciones del sistema. Cada sección te permitirá realizar acciones específicas, como ver listados, añadir nuevos registros, o editar información existente.

Tecnologías Utilizadas

Este proyecto ha sido desarrollado utilizando:

  • Flask: Microframework para el desarrollo web en Python.
  • Flask-SQLAlchemy: Gestión de la base de datos con SQLAlchemy.
  • Flask-Migrate: Herramienta para migraciones de bases de datos.
  • Bootstrap: Framework para el diseño responsive y estilizado de la interfaz de usuario.
